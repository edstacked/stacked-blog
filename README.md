# Stacked-blog

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The aim of this exercise is to

- Learn Next Js
- Collaborate

### Technologies to be use

- Tailwind
- Next Js
- gray-matter
- Markdown-it
